Anypoint Connector for MongoDB is a closed-source connector that provides a connection between the Mule runtime engine (Mule) and third-party software on a MongoDB server.

This documentation set consists of the following version documents:

| * Changed operation names, metadata, fields, and values
* Changed input sources
* Changed connection types

See the xref:mongodb/mongodb-connector-6-0-upgrade-migrate.adoc[Update and Migrate Guide] for more information.
| xref:mongodb/mongodb-connector-5-3.adoc[MongoDB Connector v5.3] +

| * Added the New object, Modified object, and Deleted object input sources, which enable you to execute a flow each time a MongoDB object is created, modified, or deleted.
